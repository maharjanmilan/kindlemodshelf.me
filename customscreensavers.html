How to Set Up Custom Screensavers on a Jailbroken Kindle

KOReader Sleep Screen: Swap Amazon’s default sleep screen for your own image folder and rotate custom art every time your Kindle sleeps.

Requirements

  • Jailbroken Kindle (KOReader requires jailbreak access)
  • KOReader installed and launching correctly
  • A custom screensaver image or folder of images

Install & Setup

  1. If your Kindle has Special Offers (ads), remove them by paying Amazon’s $20 fee or by running the Disable Ads script.
  2. Copy your custom screensaver image (or image folder) to the Kindle’s USB root (/mnt/us).
  3. Launch KOReader from your preferred launcher.
  4. Open Settings ▸ Screen ▸ Sleep Screen ▸ Wallpaper.
  5. Select Show random image from folder on sleep screen.
  6. Tap Custom images, then choose Random image folder.
  7. Pick Choose folder and navigate to your image folder.
  8. Long-press the folder name to confirm it as the source.
  9. Optional: Disable the stock message via Settings ▸ Screen ▸ Sleep Screen ▸ Sleep screen message and toggle off Add custom message.
