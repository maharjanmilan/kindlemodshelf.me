Blocking Amazon

This extension blocks amazon tracking, kindle store, and ui ads on your kindle.

Download

Latest Zip

Installation

Download the Latest Zip

Extract and then copy blockamazon to your KUAL Extensions folder (<YourKindle>/extensions/ by default).

Open KUAL and run the block or unblock function. It will take effect immediately.

Notes

This extension blocks some Amazon domains via /etc/hosts to disable the kindle store on the device.

Known Issues

If there is store cache, it will still show some ads. However new stuff is still blocked. Use our script to clear the cache to get rid of most of the remaining ads on the kindle ui.
