Quick install checklist

  1. Download the latest AdBreak package from the official release page.
  2. Confirm ads: keep the Kindle online until ads appear on the lockscreen. If no ads appear, a factory reset can sometimes help (see Troubleshooting).
  3. Enable airplane mode after ads are visible.
  4. Open lockscreen menu ‚Üí View all ads to confirm multiple Special Offers are listed.
  5. Copy .assets ‚Äì connect Kindle to PC and copy the device's .assets folder (backup the original).
  6. Unpack AdBreak and merge its files into the copied .assets folder on your PC.
  7. Run replace helper:
    • Windows ‚Äì run replace.bat
    • macOS / Linux ‚Äì run the provided terminal command to place the exploit HTML into matching ad folders (example in the release)
  8. Replace device assets: delete the original .assets on the Kindle and copy the modified folder back.
  9. Trigger exploit: unplug the Kindle, tap the modified ad, follow prompts; the jailbreak script should run at the final popup.

Backup your original .assets folder and any data before modifying anything.

Enable Special Offers (if needed)

If your device shipped without ads you can often re-enable them via your Amazon account:

  1. Amazon ‚Üí Manage Your Content and Devices ‚Üí Preferences ‚Üí Country/Region: select a supported region (US, UK, DE, FR, IT, ES, JP, CN).
  2. Add a matching billing address and a payment method (card required; usually no charge).
  3. Enable Special Offers for your device and sync ‚Äì ads should populate the lockscreen after some time.

Important: Kindle Scribe and Colorsoft cannot be made ad-enabled and are not compatible with AdBreak.
