KShips

KShips is a Battleship port for Kindle.

Download

Download KShips

Features

  • Three difficulty levels (Easy, Medium, Hard) with progressively smarter AI
  • Full Battleship gameplay with 5 ships (Carrier, Battleship, Cruiser, Submarine, Destroyer)
  • AI that intelligently attacks back and hunts down your ships
  • Ship placement with manual positioning, rotation, and random placement options

Installation

Recommended: Install through KindleForge

Or install manually:

  1. Download the latest file from GitHub Releases
  2. Unzip the downloaded file
  3. Paste both KShips/ folder and KShips.sh into the documents/ folder on your Kindle
  4. Run the scriptlet and have fun!
