KinAMP

KinAMP is a native music player developed for jailbroken Kindles (Firmware 5.16+). It features a fast, native interface optimized for e-ink displays.

Download

Download KinAMP

Features

  • Fully native, fast interface (C++ and GTK2)
  • Optimized for e-ink displays to save battery
  • Fast access to Bluetooth and frontlight settings
  • Supports MP3, FLAC, and WAV formats
  • Uses miniaudio for decoding and GStreamer backend

Installation

  1. Download the latest release from the link above.
  2. Unzip the archive to the root of your Kindle.
  3. Start KinAMP from KUAL or from the home screen.

Notes

  • No Background Playback: This is a GUI app and cannot run in the background. To listen to music while reading, SOX Media Player is a better alternative.
  • Requires Firmware 5.16 or higher.
